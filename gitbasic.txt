git and github
---------------

git operation:-
-clone: get exact copy of projet  (remote repository to local repository)
-push:  to upload new project and new updation  (local repository to remote repository)
-pull: to get new updation  (remote repository to local repository)

push
----
-add: add project to staging area from local storage(our computer storage)
-commit: add project to local repository from staging area (then push it to remote repository)

(clone & pull can done it by directly ,that is from local storage to remote repository )
